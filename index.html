Mode EPS rapide

Démarrer en moins d’une minute

  • Saisissez participants, terrains, durée de match, heure de début et pratique.
  • Activez directement Arbitre / Table si besoin, puis « Générer rapidement ».
  • Vous pouvez affiner ensuite dans les options avancées (chrono, pauses, simulation).
🏅

Sport collectif / raquette

Adapter les libellés

  • Choisissez « Sport collectif » ou « Sport de raquette » dans les options Essentiel.
  • L’app ajuste automatiquement les libellés (équipes/participants) et les tableaux.
  • Au format raquette, vous pouvez passer en match « au score cible » (11, 15 pts…).
🎯

Rôles EPS

Arbitre · Table de marque · Coach

  • Activez les rôles dans les options (ou via le mode rapide pour Arbitre / Table de marque) puis cochez ceux qui vous intéressent.
  • Par défaut, aucun rôle n’est activé. Dans le mode EPS rapide, seuls Arbitre et Table de marque sont proposés ; Coach reste disponible uniquement dans les options avancées.
  • Les équipes/participants disponibles sont affectés automatiquement et les rôles s’affichent sous chaque terrain.
🗂️

Organisation des matchs

Deux modes de planification

  • Répartition pédagogique : les rotations alternent match et repos pour faciliter les rôles EPS (arbitre, table de marque).
  • Optimiser l’utilisation des terrains : les terrains sont remplis au maximum pour réduire la durée totale du tournoi.
  • Choisissez le mode adapté selon votre créneau et votre besoin pédagogique.
🎮

Live & clôture

Piloter sur le terrain

  • Saisissez/validez les scores, suivez la rotation active/la suivante et les rôles affichés sur chaque terrain.
  • Le bouton « Terminer le tournoi » reste accessible : confirmation, puis classement final immédiat.
  • Le chrono intégré peut passer en plein écran pour afficher temps restant et matches en cours.
📺

Projection & impression

Informer élèves et arbitres

  • Depuis le live, ouvrez « Mode projection » pour afficher rotations, terrains, rôles et chrono sur TV.
  • Le bouton Imprimer génère un PDF noir & blanc (résumé, rotations, rôles, classements) prêt à distribuer.
  • Utilisez cette fiche pour briefer arbitres et table de marque avant la séance.
♟️

Ronde Suisse

Classer rapidement sans élimination

  • Chaque élève joue un match maximum par ronde.
  • Cliquez directement sur le gagnant : gagnant = 1 point, perdant = 0 point.
  • Si le nombre est impair, un élève est exempt et reçoit 1 point.
  • La ronde suivante regroupe en priorité des joueurs proches au classement.
  • Le classement live affiche points, matchs, victoires, défaites et exemptions.
  • Montée-descente : idéal pour faire tourner les terrains. Défi : idéal pour provoquer un joueur proche. Ronde Suisse : idéal pour classer vite tout un groupe avec des matchs équilibrés.
